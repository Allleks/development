**По умолчанию в QTableWidget все ячейки хранят данные как `QString` (текст)**,  даже если в ячейке хранится число или другие данные (они будут автоматически преобразованы в строку).
___
 **Типы, которые можно установить через setData()**
 Через `setData()` можно хранить любые типы данных в `QVariant`.
```cpp

QTableWidgetItem* item = new QTableWidgetItem();

// 1. Текст (по умолчанию)
item->setText("Текст");  // QString

// 2. Пользовательские данные (любой QVariant)
item->setData(Qt::UserRole, 123);           // int
item->setData(Qt::UserRole + 1, 45.67);     // double
item->setData(Qt::UserRole + 2, true);      // bool
item->setData(Qt::UserRole + 3, someObject);// указатель или сложный объект

// 3. Checkbox
item->setCheckState(Qt::Checked);  // Qt::CheckState

// 4. Иконка
item->setIcon(QIcon("path/to/icon.png"));

// 5. Выравнивание
item->setTextAlignment(Qt::AlignCenter);
```
