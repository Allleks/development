1. **`QVariant::Type`** — устаревшее перечисление только для QVariant
2. **`QMetaType::Type`** — современная система типов для всего Qt
3. **Для проверки типа QVariant** используйте:
```cpp
// Qt5: var.type() == QVariant::String
Qt6: var.typeId() == QMetaType::QString  
```
4. **Для новых проектов** всегда используйте `QMetaType`
5. **Для пользовательских типов** обязательно:
```cpp    
Q_DECLARE_METATYPE(MyType)
qRegisterMetaType<MyType>("MyType");
```
**Итог:** `QMetaType::Type` — это единый паспорт для всех типов в Qt. Если раньше у каждого типа был только "вид на жительство" в QVariant, теперь у него есть "единый паспорт", который работает везде в Qt.
