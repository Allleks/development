### **Использовать делегат когда:**
1. Нужна **кастомная отрисовка** (графики, иконки, прогресс-бары)
2. Нужен **специфичный редактор** (цветовая палитра, календарь)
3. Нужна **валидация ввода** (только числа, определенный формат)
4. Нужно **условное форматирование** (цвет текста по значению)
### **Не использовать делегат когда:**
1. Просто текст/числа - **стандартный делегат справится**
2. Нужно изменить **структуру данных** - это задача **модели**
3. Нужна **сортировка/фильтрация** - используйте **QSortFilterProxyModel**
