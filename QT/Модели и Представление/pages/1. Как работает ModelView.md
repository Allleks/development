### Основная архитектура:
```text
Данные (Data) → Модель (Model) → Представление (View) → Пользователь
```
**[[Данные]]**
* Могут быть в любом месте
**[[Модель|Модель]]**:
- Хранит данные
- Не знает о UI
- Уведомляет представления об изменениях
**[[Представление]]**:
- Отображает данные
- Не хранит данные
- Получает данные через модель
**[[Делегаты]]** (опционально):
- Контролирует отрисовку и редактирование
