В Visual Studio для C++/Qt есть несколько способов документирования:
## 1. **Doxygen стиль (рекомендуется)**:
```
/// @brief Устанавливает столбцы для отображения tooltip
/// @param columns Индексы столбцов (начинаются с 0)
/// @note Пример: setDisplayColumns({0, 1, 2}) - первые три столбца
/// @warning Индексы должны быть в диапазоне [0, columnCount()-1]
void setDisplayColumns(std::initializer_list<int> columns);
```
## 2. **Qt-стиль (тоже работает)**:
```
/*!
    \brief Устанавливает столбцы для отображения tooltip
    \param columns Индексы столбцов (начинаются с 0)
    
    Пример использования:
    \code
    model->setDisplayColumns({0, 1, 2}); // Первые три столбца
    \endcode
    
    \note Индексы должны быть в диапазоне [0, columnCount()-1]
*/
void setDisplayColumns(std::initializer_list<int> columns);
```
## 3. **XML Documentation (для IntelliSense)**:
```xml
/// <summary>
/// Устанавливает столбцы для отображения tooltip
/// </summary>
/// <param name="columns">
/// Индексы столбцов (начинаются с 0).
/// Пример: setDisplayColumns({0, 1, 2}) - первые три столбца
/// </param>
/// <remarks>
/// Столбцы должны быть в диапазоне от 0 до columnCount()-1.
/// Повторяющиеся значения игнорируются.
/// </remarks>
void setDisplayColumns(std::initializer_list<int> columns);
```
**Краткий вариант (для быстрой документации)**:
```
// Устанавливает столбцы для отображения tooltip
// Параметры:
//   columns - индексы столбцов (0-based)
// Пример: setDisplayColumns({0, 1, 2}) // первые три столбца
// Примечание: невалидные индексы игнорируются
void setDisplayColumns(std::initializer_list<int> columns);
```
**Рекомендую использовать XML Documentation (вариант 3 или 4)** - он лучше всего работает с IntelliSense в Visual Studio и понятен русскоязычным разработчикам.
