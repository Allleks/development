```sql
USE [Doodle_x_Angarsk];
GO
ALTER TABLE [Проемы]
ADD [Расположение] nvarchar(MAX) NULL;
GO
```
### Пояснение:
- **`ALTER TABLE [Проемы]`** - изменяем таблицу Проемы
- **`ADD [Расположение]`** - добавляем новую колонку с таким названием
- **`nvarchar(MAX)`** - тип данных (строка любой длины)
- **`NULL`** - поле может быть пустым (необязательное)

Если хочешь добавить поле с каким-то значением по умолчанию:
```sql
USE [Doodle_x_Angarsk];
GO
ALTER TABLE [Проемы]
ADD [Расположение] nvarchar(MAX) NULL
CONSTRAINT DF_Проемы_Расположение DEFAULT 'не указано' NULL;
GO
```
### Проверка после добавления:
```sql
-- Посмотреть структуру таблицы с новым полем
SELECT * FROM [Проемы];
-- или
SELECT COLUMN_NAME, DATA_TYPE 
FROM INFORMATION_SCHEMA.COLUMNS 
WHERE TABLE_NAME = 'Проемы'
ORDER BY ORDINAL_POSITION;
```
После добавления поля можешь вставлять в него данные:
```sql
INSERT INTO [Проемы] (devID, [Расположение])
VALUES (1, 'какое-то значение');
```
