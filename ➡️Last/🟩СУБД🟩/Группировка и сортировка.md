Как это работает:
- `from` указывает, что данные выбираются из таблицы `city`;
- `group by` объединяет все строки с одинаковым значением `federal_district`;
- `count(*)` считает количество таких объединенных строк для каждого значения `federal_district` (то есть количество городов в федеральном округе);
- `order by` сортирует результат по убыванию количества городов.

```sql
select
  колонка в таблице1 as колонка в таблице2,
  count(*) as колонка в таблице3
from название таблицы в базе
group by колонка в таблице1
order by колонка в таблице3 desc;


select
  federal_district as district,
  count(*) as city_count
from city
group by federal_district
order by city_count desc;
```
___
Как это работает:
- `where` указывает, какие строки должны попасть в результат;
- `where a = 'b'` выбирает все строки, у которых значение столбца `a` равно `'b'`;
- `where a in ('b', 'c')` выбирает все строки, у которых значение столбца `a` равно `'b'` или `'c'`;
- `where a like '%b%'` выбирает все строки, у которых значение столбца `a` подходит под шаблон `'%b%'`;
- `where a between x and y` выбирает все строки, у которых значение столбца `a ≥ x` и одновременно `a ≤ y`.

Какие города появились за последние 30 лет?
```sql
select region, city, foundation_year
from city
where foundation_year between 1990 and 2020;
```

```http
┌───────────┬───────────┬─────────────────┐
│  region   │   city    │ foundation_year │
├───────────┼───────────┼─────────────────┤
│ Ингушетия │ Магас     │ 1995            │
│ Татарстан │ Иннополис │ 2012            │

```
Или так:
```sql
select region, city, foundation_year
from city
where foundation_year between date('now', '-30 years') and date('now');
```

Сколько городов в Приволжском и Уральском округах?
```sql
select count(*)
from city
where federal_district in ('Приволжский', 'Уральский');
```
