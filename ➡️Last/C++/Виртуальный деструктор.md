#virtual #destructor
___
**Запомните:** Виртуальность деструктора наследуется по цепочке. Достаточно объявить его виртуальным в самом базовом полиморфном классе.
```cpp
#pragma once
//Базовый класс 
class Base{
	virtual ~Device() override = default;
}

//Item наследуеться еще от 
class Item : public Base
{
	// Деструктор не нужен - наследуем виртуальный от Item/
	// Или если очень хочется:
    // virtual ~Item() override = default;
}
```
Далее по цепочке наследования
```cpp
#include "Item.h"
class Device : public Item
{
public:
    QList<QString> elements;
    
    Device();
        
    // Деструктор не нужен - наследуем виртуальный от Item/
    
    // Или если очень хочется:
    // virtual ~Device() override = default;
};
```
___
Остальным классам достаточно сгенерированных компилятором деструкторов (или своих, если нужна особая очистка), виртуальность к ним придет автоматически.
