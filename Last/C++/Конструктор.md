В C++ есть **важное правило**: если вы объявляете метод/конструктор в классе, вы **обязаны** его реализовать (если он не чисто виртуальный).

```cpp
// ПЛОХО:
class MyClass {
public:
    MyClass(); // Объявлен, но не реализован
};

// ХОРОШО:
class MyClass {
public:
    MyClass(); // Объявлен
};

// В .cpp файле:
MyClass::MyClass() {
    // Реализация
}
```
